# TeamTrace Frontend Data Source Matrix

最后更新：2026-05-14

## 说明

- `真实接口`：页面主数据已直接调用当前后端接口
- `混合模式`：页面同时使用真实接口与本地缓存/预览数据
- `Hybrid 回退`：服务优先请求真实接口；本地后端不可用时自动回退演示数据
- `本地存储`：页面主数据来自浏览器本地缓存
- `占位页面`：页面结构已存在，但暂无正式接口支撑

后端支持状态说明：

- `已实现`：当前后端已有对应接口
- `部分实现`：后端只覆盖了页面所需能力的一部分
- `未实现`：当前后端没有对应页面所需接口
