PRODUCT LINE

目前系统已经形成两条主线。

第一条是课堂协作闭环，围绕教师、班级、小组任务、评分、报告、申诉和通知展开。第二条是自由协作 Beta，面向没有监管者的邀请制协作空间。

01

课堂模式

教师创建班级和任务，学生加入班级、小组协作、认领子任务、提交成果。评分与报告属于这条主线：任务评分和课程评分分离，互评限定在组内，申诉用于情况反馈和教师回复，教师端负责查看进度、处理申诉、导出报告。

02

自由协作 Beta

自由协作不做评分和绩效。它把任务、负责人、提交、接收、打回、交接和进度可视化，制造一起推进的环境。

ENGINEERING

技术栈是实际项目里正在使用的。

当前前端基于 Vue 3 与 Vite，后端基于 Spring Boot、JPA、MySQL、JWT 和邮件服务。生产运行在 Sealos，前端静态资源与 API 代理由统一入口承接。

真实架构链路

浏览器访问 TeamTrace 域名，前端负责角色工作台和页面状态，后端提供 REST API、认证、业务计算、文件上传和邮件能力，数据落在 MySQL。
